# Boundaries

This repository does not claim to prove Collatz.

It does not claim that CRC is already proven.

It does not claim that compression debt or 2-adic shadow are sufficient.

It does not claim to replace known approaches.

It defines a problem-born language and tests whether that language reduces the unresolved state of the problem.

The standard is:

    a concept survives only if it reduces the problem.

If a symbol, metric, or transformation does not help reduce the unresolved state, it should be removed.
