# Audit Report Reader Guide

Version: v4.4

This guide explains how to read the v4.4 consolidated report.

## Correct reading

Read it as a status report for a native-language research program.

## Incorrect reading

Do not read it as a proof.

Do not read it as a solution claim.

Do not read bounded absence of obstruction as global impossibility.
