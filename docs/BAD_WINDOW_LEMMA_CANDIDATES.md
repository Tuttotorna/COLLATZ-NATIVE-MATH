# Bad Window Lemma Candidates

Version: v2.2

This document introduces bad-window lemma candidates.

A bad window is the standard translation of a native local obstruction candidate.

## Lemma candidate 1: bad window equivalence

Native object:

bad window

Candidate statement:

For a finite interval I of odd blocks, the following are equivalent:

1. the average discharge over I is below log2(3);
2. cumulative debt over I is positive;
3. D(I) > 0.

Native meaning retained:

local obstruction candidate.

Status:

stable lemma candidate.

## Lemma candidate 2: bad window is local

Native object:

bad window

Candidate statement:

A bad window identifies local debt accumulation.
It does not by itself imply global obstruction.

Native meaning retained:

local danger is not native failure.

Status:

boundary lemma candidate.

## Lemma candidate 3: bad window requires recovery test

Native object:

bad window

Candidate statement:

Every detected bad window must be followed by a compensation test before obstruction status can be assigned.

Native meaning retained:

stress must be tested for repair.

Status:

stable protocol lemma candidate.

## Lemma candidate 4: many bad windows do not imply obstruction

Native object:

regeneration

Candidate statement:

The existence of many bad windows in a trajectory does not imply obstruction unless debt persists without sufficient compensation.

Native meaning retained:

regeneration is dangerous only if it avoids closure.

Status:

boundary lemma candidate.

## Lemma candidate 5: bad window obstruction boundary

Native object:

obstruction candidate

Candidate statement:

A bad window becomes relevant to obstruction only when it participates in persistent unclosed debt.

Native meaning retained:

obstruction requires self-preserving debt, not local instability.

Status:

open boundary lemma candidate.

## Current status

Bad-window equivalence is stable as a local definition.

Bad-window obstruction status remains open until persistence, shadow, regeneration, and closure are tested.
