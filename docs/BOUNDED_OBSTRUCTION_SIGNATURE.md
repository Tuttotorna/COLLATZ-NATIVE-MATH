# Bounded Obstruction Signature

Version: v2.6

A candidate must show:

    persistent debt
    non-erased shadow
    dangerous regeneration
    insufficient compensation
    no closure event
    internal admissibility

A long trajectory is not enough.

A high hardness score is not enough.

A tight positive surplus is not enough.

Repeated regeneration is not enough.

In v2.6, the scanner approximates the signature by measuring:

    bad windows
    unrecovered debt windows
    post-compensation regeneration
    dangerous regeneration
    obstruction candidate count

Dangerous regeneration is still not obstruction by itself.

The scanner may detect possible obstruction candidates.

It is not allowed to declare global proof.
