# Bounded Search Negative Result

Version: v2.6

A negative result means:

    No obstruction candidate was detected inside the tested finite domain.

It does not mean:

    Collatz is proved.
    Global closure is proved.
    Obstruction is impossible.
    All trajectories are covered.
    Finite evidence implies theorem.

A finite bounded scanner can strengthen evidence.

It cannot replace proof.

v2.6 does not do that.
