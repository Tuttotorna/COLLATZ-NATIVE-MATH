# Chain Results

v0.3 generates:

    results/cheap_regeneration_chains.jsonl
    results/cheap_regeneration_chain_summary.json

These files record finite chain diagnostics.

The analysis distinguishes:

    local cheap regeneration
    chain-compatible cheap regeneration
    broken cheap regeneration chains

Local cheap regeneration is already observed.

The real target is infinite sustainability.

A locally cheap event is not enough.

A dangerous event must be followed by another cheap regeneration while the cumulative debt remains low.

This is the first operational form of the CRC target:

    no infinite cheap regeneration chain in positive odd Collatz dynamics.
