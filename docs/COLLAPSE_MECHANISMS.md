# Collapse Mechanisms

The native language of COLLATZ-NATIVE-MATH currently contains:

    compression debt
    2-adic shadow of -1
    regeneration
    cheapness ratio
    chain compatibility
    chain collapse
    compensation surplus

v0.5 adds collapse mechanisms.

A finite compatible chain can break because:

    it pays a high compression debt
    it exhausts shadow
    it fails to regenerate cheaply
    it is overcompensated by later debt
    it terminates at 1

These mechanisms are not external explanations.

They are expressed in the same local language generated by the Collatz rule.

---

## Native proof direction

A possible proof path would be:

1. Every escaping trajectory requires an infinite cheap regeneration chain.
2. Every cheap regeneration chain must repeatedly avoid all known collapse mechanisms.
3. Avoiding all collapse mechanisms forces impossible constraints on the debt word.
4. Therefore no positive escaping trajectory exists.

v0.5 does not prove this.

It creates the first operational classification layer.
