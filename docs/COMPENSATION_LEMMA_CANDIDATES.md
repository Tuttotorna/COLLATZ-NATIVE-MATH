# Compensation Lemma Candidates

Version: v2.2

This document introduces compensation lemma candidates.

Compensation is the native repair of prior debt.

## Lemma candidate 1: compensation recovery

Native object:

compensation

Candidate statement:

Let I be a bad window.
A later interval J compensates I when the combined interval K = I union J has average discharge above log2(3).

Equivalently:

D(K) < 0

Native meaning retained:

later discharge repairs prior debt.

Status:

stable lemma candidate.

## Lemma candidate 2: local recovery is not full closure

Native object:

local recovery

Candidate statement:

If a bad window is compensated, it is locally recovered.
Local recovery alone does not imply native closure.

Native meaning retained:

recovery is not automatically obstruction erasure.

Status:

boundary lemma candidate.

## Lemma candidate 3: compensation sufficiency requires shadow test

Native object:

shadow

Candidate statement:

Compensation is not fully sufficient until the prior debt shadow is tested.

Native meaning retained:

repair must erase unresolved danger.

Status:

open lemma candidate.

## Lemma candidate 4: compensation sufficiency requires regeneration test

Native object:

regeneration

Candidate statement:

Compensation is not fully sufficient until later regeneration is tested.

Native meaning retained:

renewed danger after recovery must be checked.

Status:

open lemma candidate.

## Lemma candidate 5: compensation blocks obstruction candidate status

Native object:

obstruction candidate

Candidate statement:

If a debt structure is compensated and does not preserve shadow or dangerous regeneration, then it does not qualify as a native obstruction candidate.

Native meaning retained:

obstruction requires persistent unclosed debt.

Status:

open boundary lemma candidate.

## Current status

Compensation can be measured locally.
Full compensation sufficiency remains open until shadow and regeneration are formalized more strongly.
