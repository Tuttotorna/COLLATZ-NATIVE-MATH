# Dangerous Recurrence

Version: v3.5

Dangerous recurrence occurs when dangerous sentence forms appear repeatedly or near each other.

## Dangerous recurrence is not obstruction

Dangerous recurrence is still not proof of obstruction.

It is a native stress pattern.

## Why it matters

Dangerous recurrence tells us where Collatz repeatedly approaches the obstruction boundary without necessarily crossing it.

This is linguistically important.

It says:

Collatz can speak near-obstruction repeatedly.

But near-obstruction is not obstruction.

## Native question

When a dangerous sentence appears, what sentence tends to follow?

Possible answers:

- another dangerous sentence
- a recovery sentence
- a regeneration sentence
- an undecided sentence
- an obstruction candidate sentence

## Current boundary

If no obstruction recurrence is detected, the correct statement is:

No obstruction recurrence was detected in this bounded recurrence map.

The forbidden statement is:

Obstruction recurrence cannot exist.
