# Dangerous-Stability Invariant Candidates

Version: v3.9

Dangerous-stability invariant candidates are conserved cases where dangerous grammar remains stable.

## Meaning

This is a stronger stress signal than ordinary danger.

But it is not obstruction unless obstruction-relevant behavior is conserved.

## Required distinction

dangerous stability != obstruction

dangerous invariant candidate != theorem

bounded invariant candidate != global invariant

## Boundary

v3.9 records dangerous-stability invariant candidates without proof claims.
