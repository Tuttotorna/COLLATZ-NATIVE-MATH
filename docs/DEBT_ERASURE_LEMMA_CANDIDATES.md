# Debt Erasure Lemma Candidates

Version: v2.4

This document defines candidate lemmas for debt erasure.

## Candidate: local debt recovery

Candidate statement:

If a bad window has positive cumulative debt and a later interval restores the combined average discharge above expansion pressure, then the bad window is locally recovered.

Native meaning retained:

later discharge repairs prior debt.

Status:

stable candidate.

## Candidate: local recovery is not structural erasure

Candidate statement:

Local recovery of a bad window does not imply structural debt erasure.

Native meaning retained:

repair is not necessarily closure.

Status:

boundary candidate.

## Candidate: structural debt erasure

Candidate statement:

Debt is structurally erased only when the prior debt can no longer continue through shadow or regeneration as obstruction potential.

Native meaning retained:

debt loses obstruction power.

Status:

open candidate.

## Candidate: non-erased debt

Candidate statement:

Debt is not erased if it remains capable of preserving obstruction potential through shadow, dangerous regeneration, or insufficient compensation.

Native meaning retained:

unclosed debt remains dangerous.

Status:

open candidate.

## Boundary

Debt erasure is stronger than local recovery.

Therefore:

local recovery != debt erasure
