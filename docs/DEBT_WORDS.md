# Debt Words

A debt word is the sequence of compression debts generated by the accelerated odd Collatz dynamics.

For positive odd n_i:

    3 n_i + 1 = 2^(a_i) n_(i+1)

The debt word is:

    W(n_0) = (a_0, a_1, a_2, ...)

where:

    a_i = v2(3 n_i + 1)

Interpretation:

    a_i = 1     local escape
    a_i >= 2    local compression

The cumulative debt is:

    A_k = sum_{i=0}^{k-1} a_i

The escape threshold is:

    A_k / k <= log2(3)

A trajectory must keep its average debt low to sustain escape.
