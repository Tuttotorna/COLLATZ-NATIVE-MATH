# Entry Point Boundary

Version: v4.1

The entry point of the repository must not begin with solution pressure.

## Correct first question

What is Collatz before it becomes a conjecture?

## Incorrect first question

Can we prove Collatz?

## Correct positioning

A native structural reading of the Collatz dynamics.

## Incorrect positioning

A new proof of the Collatz conjecture.

## Required entry boundary

The first layer must state:

No solution before native language.

No proof before native description.

Not claimed solved.

Not a proof.

Not a theorem.

No global closure claim.

No global invariant claim.
