# Evidence Report Limits

Version: v2.7

This document states the limits of the v2.7 evidence report.

## Limit 1: finite source

The report reads finite v2.6 artifacts.

It cannot exceed the logical strength of those artifacts.

## Limit 2: interpretation only

The report classifies evidence.

It does not prove a theorem.

## Limit 3: no global closure claim

The report cannot claim global closure.

It can only say what the bounded scanner observed.

## Limit 4: no impossibility claim

If no obstruction candidate was detected, that is a bounded negative result.

It does not prove obstruction impossible.

## Correct use

Use v2.7 to identify:

- strongest stress cases;
- tightest surplus cases;
- most dangerous regeneration cases;
- candidates for expanded scan;
- candidates for shadow persistence instrumentation.
