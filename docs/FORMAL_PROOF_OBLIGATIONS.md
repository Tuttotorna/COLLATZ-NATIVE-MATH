# Formal Proof Obligations

Version: v3.0

This document lists what remains unproved after v3.0.

## Obligation O1: shadow erasure criterion

The project must define a theorem-level condition under which shadow is erased.

Current status:

open.

## Obligation O2: regeneration closure criterion

The project must define when regeneration is necessarily benign or compensated.

Current status:

open.

## Obligation O3: obstruction impossibility

The project must prove, not merely observe, that obstruction-preserving regeneration cannot persist indefinitely.

Current status:

open.

## Obligation O4: finite-to-general transition

The project must justify any transition from bounded evidence to general mathematical claim.

Current status:

open.

## Obligation O5: native closure theorem

The project must show that every admissible debt structure loses obstruction potential.

Current status:

open.

## Boundary

Until these obligations are satisfied, there is no proof.

formal lemma drafts created
