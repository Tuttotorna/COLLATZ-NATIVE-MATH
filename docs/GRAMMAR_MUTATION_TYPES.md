# Grammar Mutation Types

Version: v3.5

This document defines mutation types used by the recurrence map.

## repeat

The same sentence class appears again.

## family_mutation

The sentence class changes, but the family remains the same.

## dangerous_to_non_obstruction

A dangerous sentence is followed by a non-obstruction sentence.

This is important because it suggests stress without preserved obstruction.

## recovery_to_regeneration

A recovery sentence is followed by a regeneration sentence.

This is not failure.

It means repair is followed by renewed debt expression.

## regeneration_to_recovery

A regeneration sentence is followed by recovery behavior.

This is a native repair pattern.

## toward_obstruction_candidate

A transition points toward an obstruction-candidate family.

This is the most important transition class if it appears.

## cross_family_mutation

The sentence moves between different grammar families.

This may reveal local grammar instability.
