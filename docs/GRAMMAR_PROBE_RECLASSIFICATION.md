# Grammar Probe Reclassification

Version: v3.2

This document reclassifies previous computational and formal layers after the v3.1 route correction.

The earlier work remains useful.

But it must be read correctly.

## Previous scans

Previous scans are not proof attempts in the primary native layer.

They are grammar probes.

They ask:

What structures does Collatz generate under bounded observation?

## Correct interpretation

The computational layer does not say:

Collatz is solved.

The computational layer says:

These are the structures Collatz generated under bounded instrumentation.

## Native reading

The evidence is valuable because it helps learn the grammar.

It is not valuable because it pretends to close the conjecture.

## Rule

Evidence does not become proof until the native grammar has a valid bridge into standard translation.

That bridge is not yet complete.
