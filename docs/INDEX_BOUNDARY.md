# Index Boundary

Version: v4.2

The index is a navigation layer.

It is not a theory layer.

It is not a proof layer.

It is not a theorem layer.

## Correct use

Use the index to find the right artifact.

Use the index to understand the repository structure.

Use the index to avoid reading the repo as a proof-first attempt.

## Incorrect use

Do not use the index to claim:

Collatz solved.

Proof found.

Global closure established.

Global invariant established.

Obstruction impossible.

## Core rule

No solution before native language.
