# Invariant Candidate Boundary

Version: v3.9

This document defines the boundary of invariant candidates.

## Allowed

v3.9 may say:

This behavior is a bounded native invariant candidate.

## Not allowed

v3.9 must not say:

This behavior is globally invariant.

This proves Collatz.

This proves global closure.

This proves obstruction impossible.

## Why

The project is still learning Collatz's language.

A candidate is not a theorem.

A bounded invariant candidate is a native observation requiring further testing.
