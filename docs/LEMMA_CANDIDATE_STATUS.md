# Lemma Candidate Status

Version: v2.2

This document classifies the v2.2 lemma candidates.

Stable candidates:

- odd block decomposition
- discharge exponent uniqueness
- local debt formation
- cumulative debt over an interval
- bad window equivalence
- compensation recovery
- positive surplus
- zero or negative surplus
- hardness measures stress
- hardness is not obstruction
- long trajectory is not obstruction

These are stable because they follow directly from the v2.1 definition candidates.

## Boundary lemma candidates

Boundary candidates:

- local debt is not obstruction
- bad window is local
- many bad windows do not imply obstruction
- local recovery is not full closure
- positive surplus is not full closure
- finite surplus evidence is not proof

These protect the project from premature proof claims.

## Open lemma candidates

Open candidates:

- compensation sufficiency requires shadow test
- compensation sufficiency requires regeneration test
- compensation blocks obstruction candidate status
- obstruction requires persistence
- bad window obstruction boundary

These require stronger native formalization.

## Proof status

No proof claim is made.
