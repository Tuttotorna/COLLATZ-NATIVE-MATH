# Lemma Draft Boundary

Version: v3.0

This document defines the boundary of the formal lemma draft layer.

## Allowed

v3.0 may:

- state formal lemma drafts;
- define candidate hypotheses;
- define candidate conclusions;
- map every draft back to a native object;
- distinguish stable drafts from open drafts;
- identify missing proof obligations.

## Not allowed

v3.0 may not claim:

- Collatz solved;
- global closure proved;
- obstruction impossible;
- finite evidence implies proof;
- shadow always erases;
- regeneration always closes;
- local recovery equals native closure;
- positive surplus equals full closure.

## Draft is not theorem

A lemma draft is a candidate formal statement.

A theorem is a proved statement.

v3.0 contains drafts, not theorems.

## Reversibility rule

Every lemma draft must preserve the native meaning.

Required form:

native object
-> standard expression
-> formal draft
-> native meaning retained

## Failure condition

A lemma draft fails if it hides the native structure behind standard notation.

The standard layer is allowed only if it remains reversible.
