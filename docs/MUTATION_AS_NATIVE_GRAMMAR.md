# Mutation as Native Grammar

Version: v3.6

Mutation is part of the native grammar.

A language is not only its words.

A language is also how its words change.

In this repository:

- words are native objects;
- sentences are trajectory expressions;
- recurrence is repeated grammar;
- mutation is transformation between sentence forms.

## Native mutation question

The native question is:

How does Collatz transform one sentence into another?

## Important distinction

Mutation is not a proof step.

Mutation is a description step.

The project still follows:

No solution before native language.

No proof before native description.
