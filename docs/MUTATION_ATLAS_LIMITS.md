# Mutation Atlas Limits

Version: v3.6

The mutation atlas is bounded.

## Limit 1: source dependency

v3.6 depends on the v3.5 recurrence map.

It does not independently cover all trajectories.

## Limit 2: selected edges

The atlas maps selected mutation edges, especially strong and dangerous edges.

It is not the complete infinite mutation graph.

## Limit 3: no proof

The mutation atlas does not prove Collatz.

It does not prove global closure.

It does not prove obstruction impossible.

## Limit 4: dangerous is not obstruction

Dangerous mutation is still not obstruction.

Dangerous persistence is still not obstruction.

## Correct conclusion

v3.6 can say:

This bounded mutation atlas did not detect obstruction mutation.

It cannot say:

Obstruction mutation cannot exist.
