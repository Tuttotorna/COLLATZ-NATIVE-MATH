## What v4.4 does

It tells the reader what the project has built so far.

It tells the reader what Collatz has said inside the bounded native artifacts.

It tells the reader what is not being claimed.

## What v4.4 does not do

It does not add theory.

It does not prove Collatz.

It does not claim Collatz is solved.

It does not claim global closure.

It does not claim global invariance.

No solution before native language.
