# Native Closure Conditions

Version: v1.8

This document defines native closure.

Native closure is not reaching 1.

Native closure is the structural erasure of obstruction potential.

A trajectory can reach 1 only at the end of the classical process.
Native closure is earlier and structural.
It asks whether debt can still become an obstruction.

## Core definition

Native closure occurs when a generated debt structure can no longer preserve itself as an obstruction.

A native debt structure is closed when all of the following hold:

1. Debt was generated.
2. Debt shadow was carried.
3. Compensation appeared.
4. Compensation was sufficient.
5. Shadow was erased.
6. Regeneration was tested.
7. Regeneration did not preserve obstruction potential.
8. No self-preserving debt structure remained.

In short:

Native closure = obstruction potential erased.

## Closure versus terminal state

Classical terminal question:

Does the trajectory reach 1?

Native closure question:

Can the debt structure still avoid compensating discharge?

These are not the same question.

The terminal state is a classical endpoint.
Closure is a native structural condition.

## Closure is stronger than local recovery

A bad window may recover locally.
That does not automatically imply closure.

Closure requires that the recovered structure no longer carries dangerous shadow.

Local recovery says:

this debt was compensated.

Closure says:

this debt cannot continue as obstruction.

## Closure is weaker than full proof

Native closure conditions are not yet a proof of the Collatz conjecture.

They define the target that must later be translated into standard mathematical language.

## Closure condition

A native closure condition is satisfied when:

debt generated
-> debt shadow carried
-> compensation appears
-> compensation sufficiency holds
-> shadow erasure holds
-> regeneration does not preserve obstruction
-> obstruction potential is erased

## Native target

The next native target is to determine whether every admissible debt structure necessarily satisfies closure conditions.
