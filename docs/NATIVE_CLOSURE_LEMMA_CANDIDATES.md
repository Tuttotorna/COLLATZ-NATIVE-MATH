# Native Closure Lemma Candidates

Version: v2.4

This document introduces native closure lemma candidates.

These are not theorems.
These are not proof claims.
These are candidate structural statements prepared after the native method and before any full standard proof attempt.

## Core boundary

Native closure is not reaching 1.

Native closure means:

obstruction potential is erased

A closure lemma candidate must preserve this meaning.

## Candidate family 1: local recovery is not closure

Candidate statement:

If a bad window is compensated and obtains positive surplus, then the bad window is locally recovered, but full native closure is not established unless shadow erasure and regeneration testing also hold.

Native meaning retained:

local recovery does not equal obstruction erasure.

Status:

boundary candidate.

## Candidate family 2: debt erasure requires more than compensation

Candidate statement:

A debt interval is erased only when compensation removes its ability to continue as an obstruction carrier.

Native meaning retained:

debt erasure means debt loses obstruction power.

Status:

open candidate.

## Candidate family 3: shadow erasure is required for closure

Candidate statement:

Native closure requires that prior debt no longer carries obstruction-relevant influence into later trajectory structure.

Native meaning retained:

shadow erasure is necessary for full native closure.

Status:

open candidate.

## Candidate family 4: benign regeneration does not block closure

Candidate statement:

Regeneration does not block native closure if renewed debt is compensated and does not preserve obstruction potential.

Native meaning retained:

regeneration alone is not obstruction.

Status:

boundary candidate.

## Candidate family 5: dangerous regeneration requires further testing

Candidate statement:

Dangerous regeneration identifies where closure must be tested, but it is not itself obstruction unless it preserves unclosed debt through insufficient compensation and non-erased shadow.

Native meaning retained:

stress is not obstruction.

Status:

boundary candidate.

## Candidate family 6: obstruction-preserving regeneration blocks closure

Candidate statement:

If a regeneration chain preserves persistent debt, non-erased shadow, insufficient compensation, and no closure event, then native closure is not established.

Native meaning retained:

obstruction-preserving regeneration is the native counterexample target.

Status:

open candidate.

## Candidate family 7: closure candidate

Candidate statement:

A native closure candidate is present when debt is generated, compensation appears, surplus is positive, shadow is erased, regeneration is tested, and no self-preserving debt structure remains.

Native meaning retained:

obstruction potential erased.

Status:

open candidate.

## Status

v2.4 defines closure lemma candidates only.

No global closure is claimed.
No proof of the Collatz conjecture is claimed.
