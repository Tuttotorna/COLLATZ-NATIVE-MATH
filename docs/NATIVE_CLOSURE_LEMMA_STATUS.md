# Native Closure Lemma Status

Version: v2.4

This document classifies v2.4 closure lemma candidates.

## Stable boundary candidates

The following are stable as boundaries:

- local recovery is not closure;
- regeneration alone is not obstruction;
- hardness is not obstruction;
- tight positive surplus is not obstruction;
- reaching 1 is not native closure.

## Open closure candidates

The following remain open:

- debt erasure always occurs;
- shadow erasure always occurs;
- dangerous regeneration always becomes compensated or erased;
- obstruction-preserving regeneration is impossible;
- every admissible debt chain reaches native closure.

## Current status

v2.4 creates closure lemma candidates.

It does not prove closure.
It does not prove the Collatz conjecture.
It does not claim that obstruction-preserving regeneration is impossible.

## Next step

v2.5 should define an Obstruction Search Protocol.

The next correct question is:

How would we detect obstruction-preserving regeneration if it existed?
