# Native Compensation Sufficiency

Version: v1.8

This document defines compensation sufficiency.

Compensation is not sufficient merely because a bad window receives later discharge.

Compensation is sufficient only when it removes obstruction potential.

## Compensation

Compensation is later discharge that repairs a prior debt structure.

## Insufficient compensation

Compensation is insufficient when it creates only local recovery while leaving obstruction potential alive.

Insufficient compensation may show:

- tiny surplus with persistent shadow;
- repeated regeneration;
- shrinking margins;
- continued self-preserving debt structure;
- no stable erasure.

## Sufficient compensation

Compensation is sufficient when:

1. the bad window is recovered;
2. surplus becomes positive;
3. debt shadow is erased;
4. regeneration does not preserve obstruction;
5. the structure cannot remain self-preserving.

## Native sufficiency rule

Compensation sufficiency means:

recovery + positive surplus + shadow erasure + failed obstruction regeneration

## Relation to scans

Existing scans measure recovery and surplus.

v1.8 adds a stronger interpretation boundary:

recovery is not automatically closure.

To become closure, recovery must erase obstruction potential.
