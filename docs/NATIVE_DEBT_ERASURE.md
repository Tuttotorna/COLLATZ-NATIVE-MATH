# Native Debt Erasure

Version: v1.8

This document defines debt erasure.

Debt erasure is not merely a later increase in discharge.
Debt erasure means the original debt no longer has obstruction power.

## Debt

Debt is generated when discharge is insufficient relative to expansion pressure.

Debt is dangerous only if it can persist.

## Debt erasure

Debt erasure occurs when later compensation destroys the debt's ability to continue as obstruction.

A debt is erased when:

1. the bad window is compensated;
2. the combined structure returns to positive surplus;
3. the debt does not force a persistent shadow;
4. later regeneration does not preserve the same obstruction line.

## Local erasure

Local erasure means a specific bad window has recovered.

Local erasure is evidence.
It is not full closure.

## Structural erasure

Structural erasure means the debt cannot continue through shadow or regeneration.

This is the native target.

## Non-erasure

Debt is not erased if:

- it persists through shadow;
- it regenerates without adequate compensation;
- it becomes harder to erase;
- compensation margins collapse to zero or below;
- the structure remains self-preserving.

## Current evidence relation

Existing compensation scans show many cases of local debt recovery.

v1.8 does not treat those recoveries as proof.
It classifies them as evidence toward debt erasure.
