# Native Evidence Limits

Version: v1.9

This document defines what the evidence layer cannot claim.

## The evidence layer can claim

The evidence layer can claim:

- selected finite candidate sets were scanned
- bad windows were detected
- recoveries were observed
- adversarial candidates were tested
- no counterexample candidate appeared in selected scans
- hardness metrics identified different stress points

## The evidence layer cannot claim

The evidence layer cannot claim:

- global closure
- proof of Collatz
- absence of all possible obstruction
- exhaustive coverage of every admissible debt structure
- that reaching 1 is equivalent to native closure
- that local recovery is equivalent to full closure
- that hardness is equivalent to obstruction

## Boundary

The native evidence map is a bridge.

It connects computation to native structure.

It does not yet connect native structure to standard proof.

That comes only after the native-to-standard translation boundary is defined.
