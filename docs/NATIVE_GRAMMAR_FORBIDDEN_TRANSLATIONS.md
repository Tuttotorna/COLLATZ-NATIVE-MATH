# Native Grammar Forbidden Translations

Version: v3.2

This document defines forbidden translations.

The project must not collapse back into the solving posture.

## Forbidden translation 1

native grammar -> proof claim

Invalid.

The native grammar describes generated structure.

It does not prove global closure.

## Forbidden translation 2

closure -> reaches 1

Invalid.

Native closure means obstruction potential erased.

Reaching 1 is a classical terminal endpoint.

## Forbidden translation 3

dangerous regeneration -> obstruction

Invalid.

Dangerous regeneration is stress.

Obstruction requires self-preserving unclosed debt.

## Forbidden translation 4

no obstruction detected -> obstruction impossible

Invalid.

Finite negative results do not prove impossibility.

## Forbidden translation 5

hardest case -> deepest native meaning

Invalid.

A hard case is a stress case.

It may not be the deepest grammatical object.

## Forbidden translation 6

standard notation -> native understanding

Invalid.

Notation is translation.

Native understanding requires preserving the native grammar object.

## Core rule

No solution before native language.

No proof before native description.

No terminal translation before grammar.
