# Native Grammar Mutation Atlas

Version: v3.6

This document defines the Native Grammar Mutation Atlas.

The atlas asks how native sentence forms transform.

It does not ask whether Collatz is solved.

## Core rule

No solution before native language.

## Central question

How does one native sentence transform into another?

## What mutation means

A mutation is a change from one native sentence form into another native sentence form.

Mutation is not proof.

Mutation is grammar behavior.

## What v3.6 studies

v3.6 studies:

- dangerous release mutation
- dangerous persistence mutation
- recovery to regeneration mutation
- regeneration to recovery mutation
- within-family mutation
- cross-family mutation
- toward-obstruction mutation
- mutations near n0 = 9780657630

## Correct interpretation

A mutation atlas can show how Collatz changes its own sentence forms.

It cannot prove global closure.

It cannot prove obstruction impossible.

It cannot solve Collatz.

## Boundary

No obstruction mutation detected in this bounded mutation atlas is not proof that obstruction mutation cannot exist.
