# Native Grammar Recurrence Map

Version: v3.5

This document defines the Native Grammar Recurrence Map.

The map asks how native sentence forms repeat, mutate, stabilize, or disappear.

It does not ask whether Collatz is solved.

## Core rule

No solution before native language.

## Central question

Which native sentence forms repeat, mutate, stabilize, or disappear?

## What recurrence means

A recurrence is not a proof pattern.

A recurrence is a repeated grammatical behavior.

Examples:

- recovery sentence repeats
- dangerous sentence remains dangerous
- dangerous sentence mutates into recovery
- recovery mutates into regeneration
- rare sentence remains isolated
- sentence family persists across nearby starts

## What the map studies

The recurrence map studies:

- sentence-class transitions
- sentence-family transitions
- repeated sentence forms
- mutated sentence forms
- dangerous recurrence
- obstruction recurrence
- stable classes
- rare classes

## Boundary

A recurrence map is not a proof.

No obstruction recurrence detected in a bounded recurrence map is not proof that obstruction recurrence cannot exist.
