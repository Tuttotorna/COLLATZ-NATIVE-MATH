# Native Grammar Sequences

Version: v3.2

This document defines admissible native grammar sequences.

The goal is to understand what Collatz produces, not to force the classical terminal question.

## Minimal odd-block sentence

expansion -> discharge

This is the smallest native sentence.

## Debt sentence

expansion -> weak discharge -> debt

This sentence generates local danger.

## Recovery sentence

debt -> compensation -> local recovery

This sentence shows repair, but not necessarily closure.

## Shadow sentence

debt -> compensation -> shadow

This sentence shows that repair may leave structural trace.

## Regeneration sentence

debt -> compensation -> shadow -> regeneration

This sentence shows renewed danger after apparent repair.

## Benign regeneration sentence

debt -> compensation -> regeneration -> compensation

This sentence shows renewed debt that remains repairable.

## Closure candidate sentence

debt -> compensation -> shadow erasure -> regeneration tested -> no self-preserving debt -> closure candidate

This sentence indicates obstruction potential erased.

## Obstruction candidate sentence

debt -> shadow -> regeneration -> insufficient compensation -> persistent unclosed debt -> obstruction candidate

This sentence indicates possible native obstruction.

## Grammar status

These are native grammar sequences.

They are not proof claims.

They are not theorem statements.

They are descriptions of generated structure.
