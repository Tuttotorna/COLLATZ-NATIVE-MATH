# Native Grammar Stability Map

Version: v3.7

This document defines the Native Grammar Stability Map.

After recurrence and mutation, the next native question is stability.

Not:

Can we solve Collatz?

But:

What does Collatz preserve under recurrence and mutation?

## Core rule

No solution before native language.

## Stability

Stability means that a native sentence form resists mutation.

A stable grammar form is not a proof.

A stable grammar form is something Collatz keeps expressing.

## Instability

Instability means that a native sentence form changes into another form.

Instability is not failure.

It is grammar behavior.

## Dangerous stability

Dangerous stability would mean dangerous grammar persists across mutation.

This would matter because persistent danger is closer to obstruction than isolated danger.

## Dangerous instability

Dangerous instability means dangerous grammar releases, mutates, or fails to preserve itself.

This matters because it shows that danger can appear without becoming obstruction.

## Boundary

No obstruction-stable grammar detected in this bounded stability map is not proof that obstruction-stable grammar cannot exist.
