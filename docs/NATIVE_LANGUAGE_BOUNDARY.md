# Native Language Boundary

Version: v4.0

This document defines the boundary of the native-language route.

## Allowed

The project may say:

Collatz is being read as a native dynamical language.

The bounded grammar contains danger-release behavior.

The bounded grammar contains dangerous-stability behavior.

No obstruction-relevant invariant candidate was detected in the bounded artifacts.

9780657630 behaves as a grammar stress node.

## Not allowed

The project must not say:

Collatz is solved.

A proof has been found.

Global closure is established.

Obstruction is impossible.

Bounded invariant candidates are global invariants.

Danger-release behavior proves termination.

Dangerous-stability behavior is obstruction.

## Reason

The project is describing first.

Translation and proof come later, if the native language becomes strong enough.
