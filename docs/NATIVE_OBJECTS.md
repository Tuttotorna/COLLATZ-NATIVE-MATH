# Native Objects

Version: v1.6

This document defines the current native objects used by COLLATZ-NATIVE-MATH.

These objects are intentionally not introduced as standard theorem/proof language.

They are structural objects observed inside the trajectory.

## Odd block

An odd block is a native step in which an odd state is transformed by expansion and then discharged by powers of two.

In compressed form:

odd n -> 3n + 1 -> divide by 2^a until odd again

The value a is the local discharge.

## Expansion

Expansion is the multiplication pressure introduced by the 3n + 1 operation.

It is the upward force in the native trajectory.

## Discharge

Discharge is the power-of-two removal after expansion.

Discharge is measured by the exponent a.

Larger a means stronger local discharge.

## Debt

Debt is the structural burden created when discharge is too small.

A low-discharge block does not compensate enough for expansion.

Debt is not a moral or metaphorical concept here.

It is a structural imbalance.

## Cheap block

A cheap block is a block whose discharge is too small relative to the escape threshold.

The escape threshold used by the current computational layer is log2(3).

A block with discharge below this pressure contributes to local danger.

## Bad window

A bad window is a contiguous region whose average discharge is below the escape threshold.

Bad windows are not counterexamples.

They are local obstruction candidates.

## Compensation

Compensation is the later discharge that repairs a bad window.

A bad window is recovered when later blocks raise the combined average above the threshold.

## Surplus

Surplus is the remaining positive margin after compensation.

Small positive surplus means the compensation was tight.

Tight surplus is structurally important because it identifies near-obstruction behavior.

## Shadow

Shadow is the persistence of prior debt into later trajectory structure.

A trajectory can recover locally while still carrying structural shadow.

## Regeneration

Regeneration is the reappearance of dangerous structure after apparent recovery.

The native method must study regeneration because a single recovery is not enough.

## Obstruction candidate

An obstruction candidate is a structure where debt persists without adequate compensating discharge.

Current finite scans have found no unrecovered bad windows in the tested candidate sets.

That is evidence.

It is not proof.

## Native closure

Native closure would mean that every possible debt structure necessarily generates compensating discharge.

This is the central target of the native method.
