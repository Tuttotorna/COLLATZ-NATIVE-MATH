# Native Sentence Atlas Interpretation

Version: v3.4

The atlas must be interpreted through the native-language rule.

No solution before native language.

## Correct interpretation

The atlas says:

These are the sentence forms Collatz produced in the observed domain.

Some sentences show no debt.

Some show recovery.

Some show regeneration.

Some show dangerous regeneration.

No observed sentence is allowed to become a proof claim.

## Incorrect interpretation

The atlas must not be read as saying:

- Collatz is solved.
- Collatz is not solved.
- obstruction is impossible.
- reaching 1 is the native meaning.
- dangerous regeneration is obstruction.
- absence of obstruction sentence is proof.

## Why this matters

The atlas protects the project from returning too early to the classical question.

The classical question asks:

Does it terminate?

The atlas asks:

What grammar is being spoken?

## Native priority

The native priority is:

describe first
listen first
map the grammar first
translate later

The atlas is a descriptive layer.
