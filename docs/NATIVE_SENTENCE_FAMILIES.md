# Native Sentence Families

Version: v3.4

This document defines the sentence families used by the v3.4 atlas.

## no_debt_family

A sentence where no debt form is detected.

This does not mean global closure.

It only means the observed sentence did not express debt in the tested representation.

## recovery_family

A sentence where debt appears locally recoverable.

This is not full native closure.

It says that compensation appears.

## regeneration_family

A sentence where debt reappears after repair.

This is not obstruction by itself.

It is a sentence of recurrence.

## dangerous_regeneration_family

A sentence where regeneration appears with strong stress.

This is closer to the obstruction boundary, but it is still not obstruction by itself.

## obstruction_candidate_family

A sentence where persistent unclosed debt appears as a candidate form.

This would be the most important native sentence family.

Current bounded artifacts have not detected this as a confirmed sentence family.

## undecided_family

A sentence that cannot yet be classified strongly enough.

UNDECIDED is allowed.

It is better than translating too early.
