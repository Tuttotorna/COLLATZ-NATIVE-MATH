# Native Sentence Types

Version: v3.3

This document defines the native sentence classes used by the extractor.

## MINIMAL_SENTENCE

No debt sentence was detected.

## DEBT_SENTENCE

Debt was generated but the trajectory is not yet strongly classified.

## DEBT_COMPENSATION_SENTENCE

Debt was generated and compensation appeared.

## REGENERATION_COMPENSATION_SENTENCE

Debt was generated, compensated, and later regenerated in a non-obstruction sentence.

## DANGEROUS_REGENERATION_SENTENCE

Dangerous regeneration was detected.

This is stress.

It is not obstruction by itself.

## OBSTRUCTION_CANDIDATE_SENTENCE

The source artifact reports an obstruction-candidate condition.

This is not a classical counterexample by itself.

## Boundary

Sentence type is descriptive.

It does not imply proof.
