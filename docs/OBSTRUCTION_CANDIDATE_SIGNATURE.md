# Obstruction Candidate Signature

Version: v2.5

This document defines the signature required for a native obstruction candidate.

## Signature

A native obstruction candidate requires:

persistent debt
non-erased shadow
dangerous regeneration
insufficient compensation
no closure event
internal admissibility

All six are required.

## Persistent debt

Debt persists when local under-discharge is not merely compensated and erased, but continues to influence later structure.

## Non-erased shadow

Shadow is non-erased when prior debt remains obstruction-relevant after apparent recovery.

## Dangerous regeneration

Regeneration is dangerous when renewed debt appears with persistent shadow, weakening compensation, or unresolved obstruction potential.

## Insufficient compensation

Compensation is insufficient when it does not erase obstruction potential.

Local recovery is not enough.

## No closure event

No closure event means the candidate never reaches the CLOSED result type under the native closure protocol.

## Internal admissibility

Internal admissibility means the structure remains a valid trajectory structure under the native and standard translated rules.

## Boundary

A candidate is not an obstruction candidate unless all six signature components are present together.
