# Obstruction-Relevant Invariant Candidates

Version: v3.9

An obstruction-relevant invariant candidate would be the most serious native signal.

It would mean that obstruction-relevant behavior is conserved in the bounded native map.

## Status

v3.9 may detect such candidates if they exist in the bounded artifacts.

If none are detected, the correct statement is:

No obstruction-relevant invariant candidate was detected in this bounded map.

The forbidden statement is:

Obstruction-relevant invariants cannot exist.
