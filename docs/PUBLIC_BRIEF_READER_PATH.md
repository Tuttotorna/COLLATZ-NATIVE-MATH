# Public Brief Reader Path

Version: v4.5

A first-time reader should follow this order:

1. README.md
2. START_HERE.md
3. docs/V44_PUBLIC_STATUS_REPORT.md
4. results/native_audit_report_consolidation.md
5. results/native_public_research_brief.md

## Why this order

The reader first sees the corrected public position.

Then the reader sees the native-language boundary.

Then the reader sees the consolidated status.

Then the reader sees the public research brief.

## Core rule

No solution before native language.
