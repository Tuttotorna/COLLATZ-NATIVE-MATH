# Public README Shortform

Version: v4.6

This document defines the public README shortform layer.

## Purpose

v4.6 adds a short, public-facing section near the top of README.md.

The goal is to make the repository understandable in about 30 seconds.

## Required contents

- one-sentence description
- central question
- core rule
- what this is
- what this is not
- current bounded status
- link to public brief

## Core rule

No solution before native language.

## Boundary

The shortform is not theory.

The shortform is not proof.

The shortform is not a solution claim.
