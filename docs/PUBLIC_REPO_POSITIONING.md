# Public Repository Positioning

Version: v4.1

This document defines how the repository should be presented publicly.

## Short form

COLLATZ-NATIVE-MATH is a native structural reading of the Collatz dynamics.

## Medium form

This project does not begin by trying to solve Collatz. It begins by asking what Collatz is when described through the structures generated by its own transformation rule.

## Long form

Collatz is usually treated as a conjecture to solve.

This repository suspends that posture.

It treats Collatz first as a native dynamical language whose grammar can be observed, mapped, and only later translated.

## Forbidden public framing

Do not present this repository as:

- a proof of Collatz
- a solution to Collatz
- a claim that obstruction is impossible
- a claim that all trajectories have been resolved
- a replacement for standard proof

## Allowed public framing

Present it as:

- native structural reading
- grammar mapping
- bounded evidence artifacts
- non-proof native description
- pre-translation research program
