# README Shortform Reader Guide

Version: v4.6

This guide explains how to read the shortform.

## Correct reading

The shortform tells a first-time reader what the project is before they enter the technical artifacts.

## Incorrect reading

The shortform is not a mathematical proof.

It is not a theorem.

It is not a claim that Collatz is solved.

## Reader path

README shortform.

Public research brief.

Consolidated audit report.

Native language index.

Technical artifacts.
