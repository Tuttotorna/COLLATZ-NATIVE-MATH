# Recurrence Map Limits

Version: v3.5

The recurrence map is bounded.

It has strict limits.

## Limit 1: source dependency

The map depends on v3.3 and v3.4 artifacts.

It does not independently generate all trajectories.

## Limit 2: finite recurrence edges

The map studies finite local edges between observed sentences.

It does not cover the infinite grammar.

## Limit 3: neighborhood bias

Nearby starting values are not the whole grammar.

They are a local recurrence probe.

## Limit 4: no proof claim

The recurrence map does not prove Collatz.

It does not prove global closure.

It does not prove obstruction impossible.

## Correct conclusion

The recurrence map can say:

This recurrence pattern was observed in the bounded native sentence set.

It cannot say:

This recurrence pattern exhausts Collatz.
