# Regeneration Lemma Candidates

Version: v2.4

This document defines candidate lemmas for regeneration.

## Candidate: regeneration is not obstruction

Candidate statement:

The existence of a renewed bad window after compensation does not by itself imply native obstruction.

Native meaning retained:

regeneration alone is not obstruction.

Status:

stable boundary candidate.

## Candidate: benign regeneration

Candidate statement:

If renewed debt is compensated and does not preserve obstruction potential, the regeneration is benign.

Native meaning retained:

renewed stress may still be compatible with closure.

Status:

open candidate.

## Candidate: dangerous regeneration

Candidate statement:

If renewed debt appears with persistent shadow, shrinking margins, or insufficient compensation, the regeneration is dangerous and requires further closure testing.

Native meaning retained:

dangerous regeneration is stress requiring closure analysis.

Status:

open candidate.

## Candidate: obstruction-preserving regeneration

Candidate statement:

Regeneration is obstruction-preserving only when persistent debt, non-erased shadow, insufficient compensation, no closure event, and internal admissibility hold together.

Native meaning retained:

only self-preserving unclosed debt is obstruction-relevant.

Status:

open candidate.

## Boundary

regeneration != obstruction

obstruction requires obstruction-preserving regeneration
