# Shadow Erasure Lemma Candidates

Version: v2.4

This document defines candidate lemmas for shadow erasure.

## Candidate: shadow relevance

Candidate statement:

Shadow is closure-relevant only when later trajectory structure preserves obstruction-relevant dependence on prior debt.

Native meaning retained:

shadow is not arbitrary memory; it is prior debt influence.

Status:

refined candidate.

## Candidate: shadow erasure requirement

Candidate statement:

Native closure requires shadow erasure.

Native meaning retained:

prior debt must lose obstruction-carrying power.

Status:

open candidate.

## Candidate: local recovery does not erase shadow

Candidate statement:

A locally recovered bad window may still cast shadow unless later trajectory structure no longer preserves obstruction-relevant debt influence.

Native meaning retained:

local recovery is not shadow erasure.

Status:

boundary candidate.

## Candidate: shadow non-erasure

Candidate statement:

Shadow is not erased when regenerated debt remains structurally dependent on the prior deficit pattern.

Native meaning retained:

prior debt remains active as obstruction carrier.

Status:

open candidate.

## Boundary

Shadow erasure is necessary for full native closure, but v2.4 does not prove that shadow always erases.
