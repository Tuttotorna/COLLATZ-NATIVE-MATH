# Shadow Erasure Refinement

Version: v2.3

This document refines shadow erasure.

Shadow erasure is the native condition that prior debt no longer carries obstruction potential.

## Refined definition

Shadow from debt interval I is erased when later trajectory structure satisfies all of the following:

1. The original debt interval I has been locally compensated.
2. The combined compensated region has positive surplus.
3. Later bad windows, if any, are not structurally dependent on I's prior deficit.
4. Regeneration, if present, is benign rather than obstruction-preserving.
5. No self-preserving debt chain remains traceable to I.

In short:

shadow erasure = prior debt loses obstruction-carrying power

## Local recovery versus shadow erasure

Local recovery:

a specific bad window receives enough later discharge to restore positive average.

Shadow erasure:

the prior debt no longer influences later obstruction potential.

Therefore:

local recovery != shadow erasure

## Non-erasure

Shadow is not erased if:

- renewed debt repeatedly depends on the prior deficit structure;
- surplus margins shrink in a way traceable to earlier debt;
- regenerated bad windows form a self-preserving chain;
- compensation occurs but obstruction potential remains active;
- the trajectory remains governed by the earlier debt line.

## Why this matters

Native closure cannot be defined only by local compensation.
Closure requires debt erasure and shadow erasure.

Without shadow erasure, the native method cannot distinguish:

- compensated stress;
- delayed danger;
- obstruction candidate behavior.

## Status

This document defines the target.
It does not prove that shadow erasure always occurs.
