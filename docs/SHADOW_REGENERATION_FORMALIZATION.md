# Shadow and Regeneration Formalization

Version: v3.0

This document drafts formal shadow and regeneration statements.

## Draft S1: shadow signal

Native object:

shadow

Draft statement:

A prior debt interval has a shadow signal when later trajectory structure remains measurably influenced by the prior deficit structure.

Native meaning retained:

persistence of obstruction-relevant prior debt influence.

Status:

open draft.

## Draft S2: shadow signal is not obstruction

Native object:

shadow boundary

Draft statement:

A persistent shadow signal does not imply obstruction unless compensation fails and obstruction potential remains self-preserving.

Native meaning retained:

shadow stress is not equal to obstruction.

Status:

stable boundary draft.

## Draft R1: regeneration

Native object:

regeneration

Draft statement:

Regeneration occurs when a later interval again satisfies the bad-window condition after a prior compensation event.

Native meaning retained:

renewed debt after compensation.

Status:

refined draft.

## Draft R2: regeneration is not obstruction

Native object:

regeneration boundary

Draft statement:

Regeneration alone does not imply native obstruction.

Native meaning retained:

renewed debt is not automatically self-preserving unclosed debt.

Status:

stable boundary draft.

## Draft R3: dangerous regeneration

Native object:

dangerous regeneration

Draft statement:

Dangerous regeneration occurs when renewed debt appears with persistent shadow, weakening compensation, or unresolved obstruction potential.

Native meaning retained:

renewed debt that requires further closure testing.

Status:

open draft.

## Boundary

The formal layer must preserve this distinction:

regeneration != obstruction

shadow signal != obstruction

dangerous regeneration != proof of failure

obstruction requires self-preserving unclosed debt.
