# Shadow and Regeneration Status

Version: v2.3

This document classifies the status of shadow and regeneration definitions.

## Refined candidate definitions

The following definitions are now refined candidates:

- shadow;
- shadow erasure;
- regeneration;
- benign regeneration;
- dangerous regeneration;
- obstruction-preserving regeneration.

## Still open

The following remain open for proof:

- shadow always erases;
- dangerous regeneration always becomes compensated;
- obstruction-preserving regeneration is impossible;
- every admissible debt chain reaches native closure.

## Important boundary

v2.3 does not introduce theorem claims.

It refines definitions so that v2.4 can introduce Native Closure Lemma Candidates.

## Current status

The native method now has:

- native objects;
- native obstruction model;
- native closure conditions;
- native evidence mapping;
- standard definition candidates;
- local debt lemma candidates;
- refined shadow and regeneration definitions.

The next step is to express closure as lemma candidates without claiming proof.
