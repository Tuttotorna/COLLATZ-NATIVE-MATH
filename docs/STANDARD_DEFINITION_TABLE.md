| expansion pressure | log2(3) | upward pressure from multiplication by 3 |

| surplus | average discharge minus log2(3) | positive margin after repair |
| shadow | persistence of prior debt influence | memory of unresolved danger |
| regeneration | new bad window after compensation | renewed danger |
| obstruction candidate | persistent unclosed debt with no sufficient compensation | self-preserving debt |
| closure candidate | obstruction potential erased | native closure |

## Interpretation

The table is a bridge.

It is not the proof.

Each row must remain reversible.
