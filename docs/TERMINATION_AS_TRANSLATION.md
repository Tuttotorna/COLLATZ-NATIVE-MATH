# Termination as Translation

Version: v3.1

This document reclassifies termination.

## Classical terminal question

The classical question asks:

Does every positive integer eventually reach 1?

This is the standard Collatz conjecture.

It is mathematically legitimate.

But it is not the native starting point.

## Native reclassification

Termination is a translation-layer endpoint.

It is not the first meaning of the dynamics.

The native object is the generated behavior before terminal compression into the question of reaching 1.

## Native question

The native question is:

Can obstruction potential preserve itself indefinitely?

Or deeper:

What kind of debt can survive its own discharge?

This question is closer to the internal grammar of the transformation.

## The number 1

The number 1 may be the visible terminal residue.

But the native meaning may lie in the loss of obstruction potential before that terminal residue appears.

## Boundary

Reaching 1 is not native closure.

Native closure is obstruction potential erased.

The terminal state is a classical observation.

The closure mechanism is a native structural event.

## Correct use

Use termination as a later translation.

Do not use termination as the starting pressure.
