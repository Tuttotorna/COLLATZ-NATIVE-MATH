# Translation Table

Version: v2.0

This document gives the first controlled translation table.

The table is not a proof. It is a bridge between native structure and standard mathematical language.

## Table

Native object: odd block
Standard expression: odd n -> 3n + 1 -> divide by 2^a until odd
Native meaning retained: one expansion followed by one discharge event

Native object: expansion
Standard expression: multiplication pressure from 3n + 1
Native meaning retained: upward structural force

Native object: discharge
Standard expression: 2-adic removal exponent a
Native meaning retained: local reduction strength after expansion

Native object: cheap block
Standard expression: discharge exponent below escape pressure
Native meaning retained: local under-compensation

Native object: debt
Standard expression: accumulated deficit of discharge against expansion pressure
Native meaning retained: structural burden created by insufficient discharge

Native object: bad window
Standard expression: contiguous block interval whose average discharge is below escape pressure
Native meaning retained: local obstruction candidate

Native object: compensation
Standard expression: later discharge restoring positive surplus over a combined interval
Native meaning retained: repair of prior debt

Native object: surplus
Standard expression: positive margin after compensation
Native meaning retained: remaining compensation margin

Native object: shadow
Standard expression: persistence of prior deficit influence beyond local recovery
Native meaning retained: memory of unresolved danger

Native object: regeneration
Standard expression: recurrence of deficit structures after compensation
Native meaning retained: renewed danger after apparent recovery

Native object: obstruction
Standard expression: admissible non-compensating persistent deficit structure
Native meaning retained: self-preserving unclosed debt

Native object: closure
Standard expression: impossibility of persistent non-compensation under the translated conditions
Native meaning retained: obstruction potential erased

## Boundary

The standard expression is valid only if the native meaning retained column remains true.
