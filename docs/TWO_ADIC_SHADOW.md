# 2-adic Shadow of -1

Define:

    s_i = v2(n_i + 1)

This measures how close n_i is to -1 in the 2-adic sense.

If:

    s_i >= 2

then:

    n_i == -1 mod 4

and therefore:

    a_i = v2(3 n_i + 1) = 1

During an a_i = 1 escape step:

    s_(i+1) = s_i - 1

This means:

    escape consumes 2-adic shadow.

Long consecutive local escape requires high initial shadow.

Pure infinite escape corresponds to the 2-adic point:

    n = -1

which is not a positive integer.
