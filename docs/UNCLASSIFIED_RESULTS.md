# Unclassified Results

v0.6 generates:

    results/unclassified_collapse_analysis.jsonl
    results/unclassified_collapse_summary.json

Each row records a previously or potentially unclassified collapse episode and the refined labels applied to it.

The key output is:

    remaining_unclassified_count

If this count decreases, the native language is becoming more complete.

If it remains nonzero, those cases become the next mathematical target.
