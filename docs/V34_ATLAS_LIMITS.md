# v3.4 Atlas Limits

Version: v3.4

The v3.4 atlas has strict limits.

## Limit 1: source dependency

The atlas depends on v3.3 native sentence extraction artifacts.

It does not create an infinite map.

## Limit 2: bounded domain

The sentence set comes from bounded prior instrumentation.

It does not cover every trajectory.

## Limit 3: classification is provisional

Sentence families are descriptive.

They are not final mathematical categories.

## Limit 4: no proof claim

The atlas does not prove Collatz.

The atlas does not disprove Collatz.

The atlas does not prove obstruction impossible.

## Limit 5: termination remains deferred

The terminal question remains a later translation endpoint.

The native question remains prior.

## Correct conclusion

The atlas can say:

This is the grammar currently observed.

It cannot say:

This is the final truth of all Collatz trajectories.
