v4.3 Native Artifact Consistency Audit

## Why audit comes next

After the index, the repository needs a consistency audit.

The audit should check whether:

- indexed files exist
- result certificates agree with summaries
- proof boundaries are consistent
- no entry file claims solution
- no bounded artifact claims theorem
- all latest builders still run
- tests cover the latest public boundary

## Core rule

No solution before native language.
