# v4.5 Next Steps

Version: v4.5

The next recommended version is:

v4.6 Public README Shortform

## Why v4.6 comes next

After creating a public research brief, the README should receive a short public entry section.

That section should be concise, direct, and impossible to confuse with a proof-first claim.

## v4.6 target

Create a short public-facing section near the top of README.md:

- one-sentence description
- central question
- core rule
- what this is
- what this is not
- current bounded status
- link to public brief

## Core rule

No solution before native language.
