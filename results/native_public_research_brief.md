## Sections

### What this project is

COLLATZ-NATIVE-MATH is a native structural reading of Collatz dynamics. It studies the structures generated by the transformation itself before translating them into proof-oriented language.

### What this project is not

It is not a proof of Collatz. It is not a claim that Collatz is solved. It is not a theorem layer. It is not a global closure claim. It is not a global invariant claim.

### Central question

What is Collatz before it becomes a conjecture?

### What native language means here

Native language means describing Collatz through the structures produced by its own rule: expansion, discharge, debt, shadow, regeneration, compensation, closure, and obstruction.

### What the bounded artifacts currently show

The bounded artifacts show dangerous native sentence forms, danger-release behavior, dangerous stability candidates, and invariant candidates. No obstruction-relevant invariant candidate is detected in the current bounded maps.

### Why this is not a proof

The artifacts are bounded and descriptive. Bounded absence of obstruction does not imply global impossibility. Native grammar does not equal theorem. The termination question remains a later classical translation endpoint.

### Current repository integrity

The repository has a passing native artifact consistency audit, a native language index, and a consolidated native audit report.

### Where the work continues

The next work is to make the public brief easier to read, then continue native-language refinement without collapsing back into solution-first pressure.
